# 운영 방식 및 규칙

1. 해당 주에 정해진 유형 문제 풀어오기
2. 주 2회 온라인 실시간 코드리뷰
3. Discussions에는 개인적으로 푼 문제 올리기
   - 올리면 리뷰 해주기

# Commit 규칙

사이트/문제유형/문제번호 or 문제이름
ex) beakjoon/greedy/0000 or 동전교환

# PR 규칙

사이트/문제유형/문제번호 or 문제이름
baekjoon/greedy/0000 or 동전교환

### ❗️PR 시 들어가야 할 사항

1. 문제링크
2. 문제 설명
3. 문제 풀이(접근 방법)
4. 코드
5. 문제에 대한 회고
